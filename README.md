# Project Objective
## To analyze the sales, profitability, and loss patterns of Indian e-commerce platforms across states, cities, and product 
categories over a full financial year using an interactive Excel dashboard.

# Key Business Questions
Which product categories and sub-categories are most and least profitable?
Which cities and states are driving the highest revenue and profit?
What is the share of profit-making vs loss-making orders across the business?
How do sales and quantity trend month-over-month across the financial year?
What is the average order value and profit margin at category level?

# Process
Data Cleaning — Cleaned and standardized raw data using Power Query
Data Modelling — Structured data model built in Power Pivot
DAX Measures — Created calculated measures for KPIs including profit margin, average order value, and category-specific metrics
Dashboard Design — Built interactive visualizations using Pivot Tables, slicers, and charts with dynamic category filtering
